# Indra Arya - Tech & Creative Portfolio

## 📌 Deskripsi Proyek
Repository ini berisi kode sumber (source code) untuk website portofolio pribadi saya. Website ini dirancang untuk menampilkan profil profesional, keahlian teknis (IT Engineering & Data), serta karya kreatif (Videografi & Fotografi) dalam satu platform digital yang terintegrasi, responsif, dan interaktif.

## 🚀 Fitur Utama
- **Dark/Light Mode Toggle:** Tema visual yang dapat disesuaikan dengan preferensi pengguna menggunakan CSS custom properties (Variables).
- **Dynamic Project Filtering:** Sistem filter galeri kustom menggunakan JavaScript murni untuk memisahkan proyek "Tech & Dev" dan "Creative Media".
- **Advanced Animations:** Menggunakan GSAP (GreenSock) untuk animasi *hero section* dan AOS (Animate On Scroll) untuk transisi elemen yang halus.
- **Interactive Background:** Integrasi Particles.js untuk memberikan kesan modern dan dinamis pada halaman utama.
- **Custom Lightbox:** Fitur pembesaran gambar (zoom) yang dibangun tanpa *plugin* berat tambahan.
- **Fully Responsive:** Tata letak yang beradaptasi sempurna pada perangkat seluler, tablet, maupun desktop menggunakan Bootstrap 5 Grid System.

## 🛠️ Teknologi yang Digunakan
- **Struktur & Desain:** HTML5, CSS3, Bootstrap 5.3.2
- **Interaktivitas:** JavaScript (ES6+)
- **Animasi & Efek:** GSAP 3.12, AOS, Particles.js
- **Ikonografi:** FontAwesome 6.4.2
- **Tipografi:** Google Fonts (Inter & Poppins)

## 💻 Cara Menjalankan Secara Lokal
Karena website ini dibangun menggunakan teknologi statis dasar (HTML/CSS/JS), tidak diperlukan server khusus atau proses *build* (*compile*) untuk menjalankannya.

1. Clone repository ini:
   \`\`\`bash
   git clone https://github.com/indraarya77/tech-creative-portfolio.git
   \`\`\`
2. Buka folder proyek.
3. Klik ganda pada file `index.html` untuk membukanya langsung di *browser* pilihan Anda.
   *(Saran: Gunakan ekstensi "Live Server" di Visual Studio Code untuk pengalaman pengembangan yang lebih baik).*
